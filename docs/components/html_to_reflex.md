# 将 HTML 转换为 Reflex

要将 HTML、CSS 或任何设计转换为 Reflex 代码，请使用我们在 [Reflex Build](https://build.reflex.dev) 上的 AI 驱动构建工具。

只需粘贴你的 HTML、CSS 或描述你想构建的内容，我们的 AI 将为你生成相应的 Reflex 代码。

## 如何使用 Reflex Build

1. 转到 [Reflex Build](https://build.reflex.dev)
2. 粘贴你的 HTML/CSS 代码或描述你的设计
3. AI 将自动生成 Reflex 代码
4. 将生成的代码复制到你的 Reflex 应用中
