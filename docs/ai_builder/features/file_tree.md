# 文件树（File Tree）

Reflex Build 中的**文件树**让你可以**直接在浏览器中查看、组织和管理项目中的所有文件和文件夹**。它是导航应用结构的中心枢纽。

## 主要功能

### 创建文件夹
- 点击 **New Folder**（新建文件夹）按钮在当前选定的目录内创建文件夹。
- 输入名称并按 **Enter** 确认。
- 文件夹可以嵌套以分层组织你的项目。

### 创建文件
- 点击 **New File**（新建文件）按钮在当前文件夹内创建文件。
- 输入文件名和扩展名（例如 `main.py`）并按 **Enter**。
- 文件会立即显示在树中，并可在代码编辑器中编辑。

### 重命名文件或文件夹
- 右键点击文件或文件夹并选择 **Rename**（重命名）。
- 输入新名称并按 **Enter** 确认。
- 重命名会在适用的情况下自动更新项目中的引用。

### 删除文件或文件夹
- 右键点击文件或文件夹并选择 **Delete**（删除）。
- 删除的项目将被永久移除，因此删除重要文件时请小心。

### 从系统拖放
- 将文件或文件夹从你的计算机直接拖到文件树中。
- 编辑器会自动将它们导入到选定的文件夹中。
- 这适用于单个文件或整个文件夹结构，使添加资产或脚本变得快速而简单。
