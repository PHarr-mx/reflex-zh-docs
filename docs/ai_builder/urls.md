# URL

当你**直接将 URL 粘贴到 AI Builder 聊天**中时，AI 将根据你的提示词自动决定如何处理它。

你可以使用 URL 来**复制页面的设计**或**提取其内容**，而无需设置任何集成。

## 工作原理

* 如果你说类似 **"复制设计"** 或 **"使用此布局"** 的话，AI 将：

  * 对页面进行**截图**。
  * 将其用作**视觉参考**以在你的应用中重新创建 UI。
  * 允许你之后**自定义**生成的设计。

* 🪄 如果你说类似 **"获取内容"**、**"抓取此页面"** 的话，或者只是粘贴 URL 而没有提及设计，AI 将：

  * **抓取页面的内容**（文本、链接、图片、元数据）。
  * 将其作为结构化数据返回，可用于组件、工作流或 AI 操作。

## 示例提示词

* "复制此页面的设计。"
* "从此博客文章中抓取内容。"
* "从此 URL 获取所有产品详情。"
* （单独粘贴 URL）→ AI 将默认假设进行内容抓取。

## 注意事项

* **仅限公开页面：** AI 只能处理可公开访问的 URL。
* **可编辑：** 生成的设计和抓取的内容都可以在处理后进行修改。
