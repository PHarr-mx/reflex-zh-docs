# 文件（Files）

要将文件上传到 AI Builder，点击 `📎 Attach`（附加）按钮并从你的计算机中选择要上传的文件。你也可以直接将文件拖放到聊天窗口中。

本节不涵盖上传图片。查看[图片](/docs/ai/images/)以了解更多关于上传图片的信息。

```md alert
## 支持的文件类型
AI Builder 目前支持以下文件类型用于上传和处理：
1. `.pdf`
2. `.doc`
3. `.docx`
4. `.xls`
5. `.xlsx`
6. `.ppt`
7. `.pptx`
8. `.odt`
9. `.ods`
10. `.odp`
11. `.rtf`
12. `.csv`
13. `.txt`
14. `.md`
15. `.markdown`
16. `.json`
17. `.xml`
18. `.yaml`
19. `.yml`
20. `.tsv`
```

你上传的文件将自动添加到应用的 `assets/` 文件夹中，AI Builder 将能够读取和处理其内容作为提示词的一部分。

你一次可以上传的最大文件数量为 `5`。上传的最大文件大小为 `5MB`。如果你需要处理更大的文件，请考虑将它们分成更小的块或使用外部存储解决方案并通过 API 链接到它们。
