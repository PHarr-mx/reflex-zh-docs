Reflex [模型上下文协议（MCP）](https://modelcontextprotocol.io/)集成为 AI 助手和编码工具提供对 Reflex 框架文档和组件信息的结构化访问。这使得在开发 Reflex 应用时能够获得智能辅助。

Reflex MCP 服务器部署在 `https://build.reflex.dev/mcp`，并通过标准化的 MCP 工具提供对组件文档和 Reflex 文档的访问。

## 可用工具

Reflex MCP 服务器提供通过标准化 MCP 功能访问组件文档和 Reflex 文档的工具。
