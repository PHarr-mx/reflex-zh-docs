# 计费（Billing）

## 概述 

Reflex Cloud 的计费按项目每月进行。项目所有者和管理员能够查看和管理计费页面。

项目的计费由两部分组成——`席位（seats）`数量和`计算（compute）`。

## 席位

付费计划上的项目可以邀请协作者加入他们的项目。

每个额外的协作者被视为一个`席位`，并按固定的月费率收费。项目所有者和管理员可以在项目页面的设置选项卡中管理每个席位的权限和角色。

## 计算

Reflex Cloud 按秒计费，因此你只需在最终用户使用你的应用时付费。当你的应用空闲时，你不会收到费用。

有关计算定价的更多信息，请参阅[计算](/docs/hosting/compute/)页面。

## 管理计费

要管理你的计费，你可以转到项目页面 Cloud UI 中的 `Billing` 选项卡。

## 设置计费限制

如果你想为你的项目设置计费限制，你可以通过转到项目页面 Cloud UI 中的 `Billing` 选项卡来执行此操作。
